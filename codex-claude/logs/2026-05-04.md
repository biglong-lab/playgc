### 2026-05-04 [Claude]
- **動作**：🐛 修「創建隊伍」按鈕 JSON circular structure bug（使用者實測截圖回報）
- **檢查**：TS 零錯誤 ✅ / useTeamLobby.test.ts 21/21 ✅ / smoke 51/51 ✅
- **結果**：commit `dae11bcb` push main + 部署生效
- **根因**：
  - LobbyViews.tsx:388 `<Button onClick={onCreate}>` 直接綁
  - React 把 SyntheticEvent 當第一個參數傳 → handleCreateTeam(squadId) 中 squadId 變 React event 物件
  - createTeamMutation.mutate({ name, squadId: ReactEvent }) → fetch body JSON.stringify(ReactEvent) → circular 失敗
- **修法（兩道防線）**：
  1. LobbyViews.tsx:388 — `onClick={onCreate}` → `onClick={() => onCreate()}`
  2. useTeamLobby.ts:391 — handleCreateTeam 加 type guard（`typeof squadId === "string"` 才用、否則 undefined）
- **判斷**：
  - 線上問題、玩家無法組隊、優先修
  - 多人活動立即可恢復使用
  - type guard 同時防未來其他呼叫處再踩同雷
- **建議下一步**：
  - 使用者重測組隊 → 創建隊伍應該成功
  - sweep 其他 mutation handler 是否有類似 onClick={fn} fn 接參數 的 pattern（暫無發現）
- **檔案**：2 修改（LobbyViews.tsx + useTeamLobby.ts、共 4 行 +）
- **任務 ID**：Team-Create-JSON-Fix
