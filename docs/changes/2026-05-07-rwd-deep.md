## 新增 RWD utility class（index.css）

```css
/* 拍照預覽 / 紀念照固定比例 */
.aspect-photo { aspect-ratio: 4 / 3; }
.aspect-photo-portrait { aspect-ratio: 3 / 4; }

/* 玩家頁主容器（手機自然全寬、桌面置中限寬）*/
.container-player {
  max-width: 42rem;
  margin: 0 auto;
  padding: 0 1rem;
}
@media (min-width: 768px) {
  .container-player { padding: 0 1.5rem; }
}

/* 防止超長詞撐爆容器（中英混排常見）*/
.text-balance-safe {
  word-break: break-word;
  overflow-wrap: anywhere;
}
```

未來新元件直接用這些 class、不用重複寫 RWD 邏輯。
