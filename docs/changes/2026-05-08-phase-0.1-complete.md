## 2. 完成清單

### Server（`server/routes/admin-multi-sessions.ts`）

新增 `GET /api/admin/multi-sessions` endpoint：
- 列 admin 場域內所有 `status='playing'` 的 sessions
- super_admin 看全場域、其他 admin 限自己場域
- batch query 4 步（避免 N+1）：
  1. active sessions + games join
  2. teams（依 gameId batch）
  3. team_members（active、依 teamId batch）
  4. player_progress（5 分鐘內 updated 視為 online）
- 回傳 lightweight：sessionId / gameTitle / teamCount / total/online/offline members
- 既有 `GET /api/admin/multi-sessions/:gameId/state` 不動

### Client

新增頁面：
- `client/src/pages/admin/AdminMultiSessions.tsx`（~290 行）
  - 5 秒自動 refresh（refetchIntervalInBackground: false 省電）
  - 主列表：每張 SessionCard 顯示 4 個 stats（teamCount / total / online / offline）
  - 在線比 < 60% → 紅色 badge 警示
  - 點卡片展開 → SessionDetail（呼既有 endpoint 拿詳細資料）
    - 每隊：成員 online 狀態 / progress 頁數 / 進度 % / 分數
    - 隊長 badge
    - 最近 3 條 team_game_states snapshot

修改：
- `client/src/config/admin-menu.ts` — 加「📡 即時連線監控」項（在「進行中場次」後）
- `client/src/App.tsx` — 加 `AdminMultiSessions` lazy import + `/admin/multi-sessions` route
