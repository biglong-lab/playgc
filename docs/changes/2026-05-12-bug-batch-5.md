## 修改檔案（19 個）

### useSessionManager 重構（#1 + #5）
- `client/src/pages/hooks/useSessionManager.ts`
  - 加 `pendingDecision` / `userDecided` state
  - 偵測有實質進度的 existingSession 時 pendingDecision=true、不自動 restore
  - 玩家選「繼續」呼叫 `confirmContinue()` / 選「重新開始」呼叫 `resetAndCreateNew()`
  - resetAndCreateNew 加 `setForceNewSession(true)` + invalidate query

- `client/src/pages/GamePlay.tsx`
  - pendingDecision 時 early return 顯示 dialog（蓋遊戲頁面）
  - 用 `existingProgressInfo` 算 dialog 顯示的進度（pendingDecision 時 currentPageIndex 還沒 restore）

### #3 廣修 16 個元件 reward fallback
全部 `?? <number>` 改 `?? 0`：
- PhotoTeamFlow / PhotoTeamGather / ConditionalVerifyPage / PhotoBeforeAfterFlow / GpsMissionPage (×2) / PhotoBurstFlow / PhotoArStickerFlow / TimeBombPage / LockPage / TextVerifyPage / MotionChallengePage / PhotoCompareFlow / MemoryMatchPage / PhotoSpotFlow / PhotoOcrFlow / useQrScanner

### #2 LocationSelect 顯示對應 page
- `client/src/components/shared/LocationSelect.tsx`
  - 加 `allPages` prop
  - `findMatchingPage`：用 lat/lng 容差 0.0001（約 11m）匹配
  - 顯示「#N · pageName」+ pageType badge

- `client/src/pages/game-editor/ConditionalVerifyEditor.tsx`
  - 加 `allPages` prop、傳給 LocationSelect

- `client/src/pages/game-editor/PageConfigEditor.tsx`
  - 傳 allPages 給 ConditionalVerifyEditor

### #4 GPS 箭頭朝向修正
- `client/src/hooks/useCompassHeading.ts`（新檔）
  - iOS：`webkitCompassHeading`（順時針）+ requestPermission（user gesture）
  - Android：`alpha`（逆時針 → 翻正為順時針）
  - 跨平台統一回 0-360 順時針 heading

- `client/src/components/game/gps-mission/GpsMissionMap.tsx`
  - bearing = `(bearing_to_target - device_heading + 360) % 360`
  - 玩家轉身、箭頭跟著轉
  - 不支援指南針的裝置 fallback 用絕對方位
  - iOS user gesture 偵測（pointerdown / touchstart）
