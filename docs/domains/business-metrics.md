**狀態定義**：
- 🟢 active — 訂閱中或一次性已付款
- 🟡 trial — pilot 試用、未付費
- 🔴 churned — 取消 / 退訂
- ⚫ pending — 議價中、未成單

## MRR 計算公式

```
MRR = SUM(active 訂閱客戶 × 月費)
ARR = MRR × 12

一次性方案不計入 MRR（但計入累計營收）
委辦方案依合約期攤提（NT$ 200K / 6 個月合約 = NT$ 33.3K MRR）
```
