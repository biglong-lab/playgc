## 🔍 共通確認項目（測完所有任務後）

### 速度 & 穩定性
- [ ] 沒有 504 / 502 錯誤
- [ ] 沒有 console.log 紅色 error
- [ ] 沒有「載入中」轉圈卡 > 30 秒
- [ ] 玩家退出再進入仍能繼續流程

### UX 品質
- [ ] success 變體訊息**多元**（不重複看到同一句）
- [ ] fail 變體訊息**鼓勵性**（不打擊）
- [ ] hint 出現時機合理（玩家卡關才出現）
- [ ] 錯誤訊息**全中文**（無英文殘留）

### 資料正確性（管理後台確認）
- [ ] `/platform/ai-center` → 用量總覽顯示新呼叫
- [ ] `/admin` → AI 服務用量數字增加
- [ ] `ai_usage_logs` 有對應 `success=true` 紀錄
- [ ] `player_event_logs` 有 `page_complete` 事件
- [ ] `variant_feedback` 玩家按讚 / 踩有寫入

### 安全 / Fallback
- [ ] 故意斷網拍照 → 友善錯誤訊息（不白屏）
- [ ] OpenRouter 限額 → 自動 fallback 到次選模型
- [ ] AI timeout → 不卡死，可放棄該任務或重試
