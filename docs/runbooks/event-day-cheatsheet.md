# 活動當天小抄（業務 / admin 印一份帶現場）

> 列印格式：A4 一張、單面、隨身帶、避免活動當天慌亂

---

## 🎯 開始前 30 分鐘

### 設備檢查
- [ ] 投影機接好筆電（HDMI / USB-C）
- [ ] Wi-Fi 正常（手機 + 筆電都能連）
- [ ] QR 已列印 + 貼在指定位置
- [ ] LINE 通知群組已建立

### 大螢幕端啟動
1. 筆電瀏覽器開：`https://game.homi.cc/host/<sessionId>?token=<hostToken>`
2. F11 全螢幕
3. 投影機顯示
4. 確認看到對應元件畫面（拍立得牆 / 排行榜 / ...）

### 玩家測試
- 用自己手機掃玩家 QR
- 確認進入 `/play/<sessionId>` 或 `/g/<slug>`
- 確認能看到對應元件
- 確認大螢幕 / 玩家手機間互動正常

---

## 🎬 活動進行中

### 多元件切換策略

如果你的情境有多個 host 元件（婚禮 = 拍立得 + 簽名簿 + emoji），
切換時機：

| 時段 | 大螢幕投影 | 操作 |
|------|---------|------|
| 賓客抵達 | 拍立得紀念牆 | 玩家上傳合照、留言 |
| 儀式進行 | 暫停（投影一般 slide）| 不投我們的 |
| 進場 / 戒指交換 | Emoji 池 | 全場應援 |
| 餐後 | 簽名簿 | 留言給新人 |
| 活動結尾 | 拍立得回顧 | 把整晚留言播放 |

切換方式：
- 換另一個瀏覽器 tab 開另一個 hostUrl
- F11 全螢幕

## 🚨 常見緊急狀況

### Q：玩家手機顯示「session 已結束」？

> 90% 是 hostToken 過期（12 小時了）。
> 立即 LINE 服務商重建場（5 分鐘）
> 客戶不必換 QR、玩家 URL 不變

### Q：大螢幕黑屏 / 失去連線？

> 1. 重新整理頁面（F5）
> 2. 仍黑 → 檢查 Wi-Fi
> 3. 仍黑 → LINE 服務商
> 4. 玩家手機端不受影響、繼續玩

### Q：玩家留言不顯示？

> 1. 看大螢幕 + 玩家手機都是同一個 sessionId
> 2. 玩家確認有按「送出」
> 3. 仍沒顯示 → LINE 服務商

### Q：投影機顯示玩家匿名 / 不雅留言？

> hostMode 端可即時刪除（W11+ 規劃功能）
> 目前 W9：admin LINE 告知 → admin 從後台刪 page
> 緊急可 F11 退出全螢幕、暫切到一般 slide

### Q：Wi-Fi 不穩定多人斷線？

> 玩家端會自動重連（30 秒內）
> 重連後互動恢復
> 持續斷線 → 切備援 4G 熱點
