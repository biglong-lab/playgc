# 多活動預約 + POS 工作站 — 業主操作手冊

> 對象：場域 admin / 現場工作人員
> 範圍：建立活動 → 玩家預約 → 現場報到/收款/核銷 → 看營收
> 更新日期：2026-05-18

---

## 一、業主第一次設定（admin）

### 步驟 1：建立活動

1. 用 admin 帳號登入 https://game.homi.cc/admin
2. 左側選單 → **遊戲中心** → **🎯 活動管理**
3. 點右上「新增活動」
4. 填寫：

| 欄位 | 範例 | 說明 |
|------|------|------|
| 活動名稱 | 賈村射擊體驗 | 顯示在卡片 + LINE 通知 |
| Slug | shooting | URL 用、英數和 `-`、建後不可改 |
| 封面圖 URL | `https://res.cloudinary.com/...` | LINE Flex hero + 預約頁 hero |
| 卡片簡介 | 「60 分鐘專業教練帶領」 | 列表顯示 |
| 詳細說明 | （多行）含護具/彈藥/集合點 | 詳細頁顯示 |
| 集合地點 | 賈村集合點、靶場入口 | 顯示在卡片 + 預約成功訊息 |
| 單人價 | 800 | NT$、會自動 ×人數 算總價 |
| 時長 | 60 | 分鐘 |
| 單梯人數 | 4 | 一個時段最多接幾人 |
| 付款模式 | 現場付款 | onsite / online / both |

5. 儲存後活動即上線（業主可看到玩家端立刻有卡片）

### 步驟 2：建議建立的活動模板（賈村）

| 活動 | 價格 | 時長 | 人數 | 模式 |
|------|------|------|------|------|
| 射擊體驗 | NT$800 | 60 分 | 4 人 | 現場 |
| 水彈對戰 | NT$1,200 | 90 分 | 8 人 | 現場 |
| 實境闖關 | NT$500 | 45 分 | 6 人 | 現場 |
| 文化導覽 | NT$300 | 30 分 | 10 人 | 現場 |

> 💡 線上付款模式（online / both）等業主有 Recur.tw / Stripe / LinePay 商戶帳號後再啟用。

## 二、玩家端流程（你不用做、給你了解）

1. 玩家從 LINE / Web 進入 `https://game.homi.cc/book/<場域>`
2. 看到活動卡片列表（業主建立的所有啟用中活動）
3. 點卡片 → 進入該活動專屬預約頁
4. 選日期 + 時段 + 人數 + 留言 → 點「立即預約」
5. 收到 LINE Flex 卡片：
   - 活動封面 + 名稱
   - 預約成功 ✅
   - 時間 / 人數 / 金額 / 預約碼
   - 大字「📱 出示此預約碼給現場工作人員以核銷」
   - 「查看預約詳情」按鈕
6. 點卡片進預約完成頁、看到大 **QR Code**（給現場掃用）

## 三、現場工作人員流程（POS）

### A. 開機

1. 用手機 / 平板開啟 https://game.homi.cc/pos
2. 用場域 admin 帳號登入
3. 看到「POS · 賈村競技場」工作站首頁

### B. 首頁 Dashboard

立刻看到 4 個大字統計：
- 今日預約 N 組
- 已到場 M 組
- 已收款 NT$X
- 收款 N 筆

下方「下個 30 分鐘」清單 — 顯示準備到場的預約。

### C. 玩家到達 → 掃 QR 報到

1. 按「掃描 QR」大按鈕
2. 玩家出示手機 LINE 訊息上的 QR
3. 對準框內 → 自動辨識
4. 確認頁顯示：玩家姓名 + 活動 + 時間 + 金額
5. 點「標記到場」按鈕
6. 同時可點「現場收款」進收款頁

> 💡 沒有相機 / 不順？可改用「手動輸入預約碼」（6 字元短碼如 C3WJAZ）

### D. 收現金

1. 從 QR 確認頁、或「今日預約」清單點該預約的「收款」按鈕
2. 系統自動帶入金額（玩家預約時的金額 × 人數）
3. 確認金額（可改）
4. 點現金按鈕
5. 點「確認收款 NT$X」
6. 看到綠色超大字「NT$X 已收款」確認

**散客現場購買**（沒預約直接來）：
- 從首頁按「現金收款」
- 自己填金額 + 客名（選填）
- 完成

### E. 核銷券（玩家拿券來折抵 / 兌換）

> 💡 Phase 4-E 簡化版、目前可掃預約碼。Phase 5 完整折抵流程上線後、玩家收到的券（從遊戲發券）可直接掃 → 折抵 / 兌換。

1. 從首頁按「券核銷」
2. 掃券 QR 或手輸券碼
3. 看到券資訊（哪張券、面額、玩家）
4. 點「確認核銷」

### F. 今日清單

- 從首頁按「今日預約」
- 看所有今日預約、按活動分組、有 3 個按鈕（報到 / 收款 / 未到）
- 可搜尋姓名或預約碼

### G. 班次結束 → 看今日小結

- 從首頁按「小結」
- 看到漸層大字總計：今日總收款 NT$X
- 按活動分組（哪個活動收多少）
- 交易明細逐筆顯示
- 60 秒自動刷新

## 四、業主月底看營收

1. https://game.homi.cc/admin
2. 左側選單 → **財務中心** → **營收總覽**（如已啟用 payment 模組）
3. 看到三大來源：
   - 遊戲收入（線上購買 / 兌換碼）
   - 對戰收入（水彈報名）
   - **POS 收入**（現場收款、含按活動切片）

## 五、常見問題

### Q1：玩家掃 QR 後我看不到他的資料？

A：可能 booking 是其他場域的、或 QR 過期。讓玩家直接給預約碼（6 字元）手動輸入。

### Q2：相機開不起來？

A：iOS 14.3+ / Chrome 才支援自動 QR 辨識。舊機型請用「手動輸入預約碼」。

### Q3：散客現場買、要不要建活動？

A：不用、`/pos/checkout` 直接手填金額即可、不影響營收統計（會列為「未分類」）。

### Q4：玩家預約後沒收到 LINE？

A：1. 確認場域 LINE 設定（admin → LINE 設定）；2. 玩家有沒加 LINE 官方好友。

### Q5：可以退款嗎？

A：Phase 4-A 退款管理在計畫上、需業主有 Recur.tw / Stripe 商戶帳號後才能啟用線上退款。線下退款請手動在 `pos_transactions` 加備註紀錄。

### Q6：要怎麼指派現場人員只能用 POS、不能改設定？

A：建立角色「pos_operator」、勾以下 permissions：
- `pos:view` `pos:scan` `pos:checkout` `pos:voucher_redeem`
- `booking:view_today` `booking:mark_attended`
- **不要**勾 `game:edit` / `field:edit`

### Q7：活動建錯了想刪除？

A：admin → 活動管理 → 點該活動的「🗑️」按鈕 → 「停用」（不會真刪、玩家端就看不到）。

## 六、安全規則（必讀）

1. ❌ 不要刪 admin 帳號的權限（會鎖住自己）
2. ❌ 不要把現場 POS 機器登入超級管理員帳號（建專屬 pos_operator 帳號）
3. ❌ 收款後出現任何異常、保留交易紀錄、不要直接改 DB
4. ✅ 每天打烊前看「小結」確認總收款 = 現金抽屜實收
5. ✅ 每月底檢查 `/admin/revenue` 確認所有收入來源
